.. admonition:: Překonané API
   :class: warning

   JIP/KAAS legacy API je určeno pro stávající systémy, které jsou integrovány vůči JIP/KAAS, aby jim umožnilo snadný přechod na CAAIS. Jedná se o proprietární, konzervované a překonané API. Při jeho použití snadno narazíte na potíže, které v moderních API řešit nemusíte.
   
   **Důrazně nedoporučujeme používat JIP/KAAS legacy API pro nové systémy.**
